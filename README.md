# Mini Python Calculator

A simple Python calculator that lets you perform basic arithmetic operations:  
**Addition (+), Subtraction (-), Multiplication (*), Division (/)**.  

It also includes an optional tutorial for first-time users.

---

## How to Use

1. Run the script:

```bash
python calculator.py
```
2. Follow the prompts:
```text
Do you want tutorial? (Yes/No)
Write your first number:
Write your second number:
Write your action (+, -, *, /):
```
3. Get the result instantly!

---

## Features

- Simple and beginner-friendly
- Optional tutorial included
- Handles division by zero gracefully
- Minimal and clean code

---

## Example

```text
Hi guest, this is a calculator! You can calculate anything you want. Enjoy!

Do you want tutorial? (Yes/No) Yes
Ok here is:
First you write me the first number, then you write the second number, and finally you must write the action you want to perform with these numbers.

Write your first number: 5
Write your second number: 3
Write your action: +
Result: 8
```
